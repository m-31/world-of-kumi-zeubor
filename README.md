## Program execution

Build the fatJar with `./gradlew shadowJar` or `gradlew.bat shadowJar`.
Then you can start the program with

    java -jar build/libs/world-of-kumi-zeubor-*-all.jar 

or

    java -jar build\libs\world-of-kumi-zeubor-*-all.jar 


You must add some command line parameters like

    gui 4
